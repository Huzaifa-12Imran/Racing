# Neon Drift 3D

A fast-paced, low-poly 3D drifting arcade racing game built entirely in the browser using HTML5, JavaScript, and Three.js! 

## 🏎️ Features
- **Dynamic Physics:** Fun, arcade-style drifting mechanics with speed boosts and combos.
- **Procedural Audio Synthesis:** Fully synthesized engine sounds, tire squeals, crashes, and coin chimes using the Web Audio API. No external audio files needed!
- **Juicy Camera Effects:** Dynamic Field-of-View (FOV) warping and anime-style speed lines when hitting top speeds.
- **Collectibles:** Risk/reward drifting with golden rings spawning on the edges of sharp corners.
- **Day/Night Cycle:** Seamlessly interpolating track lighting from vibrant daytime to glowing neon nights.
- **Toon Shading:** Custom shader materials that give the game a vibrant, cartoon low-poly aesthetic.

## 🛠️ Technology Stack
- **Core:** JavaScript, HTML, CSS
- **3D Engine:** [Three.js](https://threejs.org/)
- **Build Tool:** Vite

## 🚀 How to Run Locally
1. Ensure you have Node.js installed.
2. Clone this repository.
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the local development server:
   ```bash
   npm run dev
   ```
5. Open your browser to the local address provided by Vite (usually `http://localhost:5173`).

## 🎮 Controls
- `W` / `Up Arrow`: Accelerate
- `S` / `Down Arrow`: Brake / Reverse
- `A` / `D` or `Left` / `Right Arrow`: Steer
- `Shift`: Hold while turning to Drift! Release to get a speed boost.

*Drive fast, drift hard, and don't crash!*
